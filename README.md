# Which State Am I? – Fast Forward to 2050

## Overview

This project was created for the Hack2Skill × Google for Developers "Which State Am I?" challenge.

The theme chosen is **Fast Forward to 2050**.

The image presents a futuristic vision of an Indian state where artificial intelligence, sustainable development, smart infrastructure, cultural heritage, and human-centered innovation coexist. The goal is to encourage viewers to identify the state through visual clues embedded in the artwork without explicitly revealing its name.

## Theme

Fast Forward to 2050

## Concept

The artwork imagines how a state known for innovation, technology, heritage, and culture could evolve by the year 2050 while preserving its identity and traditions.

## Features

* AI-generated futuristic visualization
* Sustainable smart-city concepts
* Heritage preservation through technology
* Human-centered innovation
* Interactive state-guessing challenge
